Data Binding

Sincronizacion automatica entre datos JavaScript y la UI sin tocar el DOM.

Donde se usa

En los templates de Polymer mediante [[ ]] y {{ }} para mostrar datos y sincronizar formularios.

Como se usa

{{prop::input}} es two-way: si el usuario escribe, la propiedad JS se actualiza sola automaticamente.

Que hace

Mantiene sincronizados los datos y la UI, actualizando automaticamente todos los elementos vinculados cuando cambia una propiedad.

vs HTML5 puro

En HTML5 hay que capturar eventos y actualizar el DOM a mano. En Polymer es declarativo y automatico.

HTML5 puro
<input id="inp">
<p id="texto"></p>

input.addEventListener('input', () => {
  texto.textContent = input.value;
});
Polymer — Two-way Binding
<!-- HTML -->
<input value="{{nombre::input}}">
<input value="{{nombre::input}}">
<p>Hola, [[nombre]]</p>

// JavaScript
static get properties() {
  return {
    nombre: { type: String, value: '' }
  };
}

Properties

Sistema declarativo con tipo, valor, computed y observer.

Donde se usa

En static get properties(). Toda propiedad reactiva que deba actualizar el DOM al cambiar se declara ahi.

Como se usa

Cada propiedad es un objeto con claves como type, value, observer y computed. Polymer se encarga del resto.

Que hace

Cuando una propiedad cambia, Polymer actualiza el DOM, recalcula los computed y dispara los observer automaticamente.

vs HTML5 puro

En vanilla necesitas getters/setters manuales, llamar render() a mano y recalcular derivados en cada cambio. Mucho codigo repetido.

HTML5 puro
<!-- HTML -->
<p id="num">0</p>
<button id="btnMas">+ Sumar</button>

// JavaScript
let contador = 0;
const num = document.querySelector('#num');

function render() {
  num.textContent = contador; // DOM manual
}
document.querySelector('#btnMas')
  .addEventListener('click', () => { contador++; render(); });
Polymer
<!-- HTML -->
<p>[[contador]]</p>
<button on-click="_mas">+ Sumar</button>

// JavaScript
static get properties() {
  return {
    contador: { type: Number, value: 0 }
  };
}
_mas() { this.contador++; }

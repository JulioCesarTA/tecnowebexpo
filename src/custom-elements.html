Custom Elements

Define tus propias etiquetas HTML reutilizables con comportamiento y estilos propios.

Donde se usa

Cuando queres crear una etiqueta HTML propia que encapsule logica, estilos y template reutilizable en cualquier parte de la app.

Como se usa

Extendiendo PolymerElement y llamando customElements.define('mi-elem', MiClase). Despues usas <mi-elem> como cualquier etiqueta HTML.

Que hace

Registra un nuevo elemento en el navegador con su propio Shadow DOM, estilos aislados y propiedades reactivas.

vs HTML5 puro

En vanilla tenes que gestionar manualmente el Shadow DOM, el template, los atributos y el ciclo de vida. Polymer lo hace declarativo.

HTML5 puro
class MiTarjeta extends HTMLElement {
  constructor() {
    super();
    this.attachShadow({ mode: 'open' });
  }
  static get observedAttributes() {
    return ['nombre'];
  }
  attributeChangedCallback(attr, old, nuevo) {
    this.shadowRoot.innerHTML =
      `<p>Hola, ${nuevo}</p>`;
  }
}
customElements.define('mi-tarjeta', MiTarjeta);
Polymer
class MiTarjeta extends PolymerElement {
  static get template() {
    return html`<p>Hola, [[nombre]]</p>`;
  }
  static get properties() {
    return { nombre: { type: String } };
  }
}
customElements.define('mi-tarjeta', MiTarjeta);
